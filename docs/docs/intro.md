# NS Tools 101

The tool Atlas is an interactive map of tools and projects in the Network School community. Browse what NS members are building, submit your own project, or request tools you'd like to see.

:::tip Explore the Atlas
Head to the [live Atlas](/) to see the full interactive canvas — or check [/requests](/requests) to suggest a tool you'd like built.
:::

## Pages

| Route        | Description                                                                              |
| ------------ | ---------------------------------------------------------------------------------------- |
| `/`          | Main atlas — interactive canvas (desktop) or card list (mobile) of all approved projects |
| `/requests`  | Community wishlist — suggest tools and upvote ideas from other members                   |
| `/graveyard` | Archive of NS tools that have shut down                                                  |
| `/data`      | Charts and stats about the ecosystem                                                     |

## Categories

Projects are organized into categories:

- **Networks** — community and social platforms
- **Coworking** — shared workspaces
- **Events** — meetups, conferences, gatherings
- **Media** — content, publishing, news
- **Education** — learning platforms and resources
- **Local VCs** — local venture capital
- **Global VCs** — international venture capital
- **Accelerators** — tool programs
- **Corporate** — corporate ventures and partnerships
- **Transport** — mobility and logistics

:::info Custom categories
Members can also propose custom categories when submitting a project. New categories get auto-assigned a unique color.
:::
