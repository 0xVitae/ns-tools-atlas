# Requesting a Tool

Don't have a project but want to see something built? You can suggest ideas for tools the NS community needs.

## How it works

1. Head to [`/requests`](/requests)
2. Describe the tool or project you'd like to see
3. Submit your request

Once submitted, your request is visible to the entire community. Other NS members can **upvote** requests to signal demand — the most popular ideas rise to the top.

## What makes a good request

- **Be specific** — "A tool that tracks NS event attendance" is better than "something for events"
- **Explain the problem** — Why does the community need this? What gap does it fill?
- **Check existing tools first** — Browse the Atlas to make sure something similar doesn't already exist

## What happens next

Requests with strong community support help signal to builders what's worth working on. If you see a request you'd like to tackle yourself, go for it — [submit it as a project](/docs/submitting-a-project) when it's ready.
