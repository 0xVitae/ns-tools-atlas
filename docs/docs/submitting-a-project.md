# Submitting a Project

Any NS member can submit their project to the Atlas. Before you begin, make sure you meet the following prerequisites.

## Prerequisites

### 1. Sign in

You must be signed in to submit a project. Authentication is handled through your NS account and Discord.

### 2. Be an NS member

You need to be a verified Network School member. During submission, you'll be asked to verify your NS profile URL to confirm you're part of the community.

:::info Profile URL format
Your profile URL looks like: `https://ns.com/your-username`
:::

### 3. Set up NS Authentication with Discord

The Atlas uses NS Auth to verify your NS identity via Discord. If you haven't set this up yet, the quickest way is to use the **Claude Code plugin** — just run:

```
/ns-auth
```

This skill walks you through the full NS Auth integration step by step, including Discord verification.

For manual setup or more details, see the [Claude Code Plugin docs](/docs/claude-code-plugin).

## How to submit

1. Open the Atlas at `/`
2. Click the **Add Project** button
3. Fill in your project details and submit
4. Agree to form on ns.com/platform

:::info Required fields
**Name**, **Category**, **URL**, and **Description** are all required. You can add additional details like screenshots, team members, and plans at any time after submission.
:::

## Project fields

| Field               | Required | Description                                     |
| ------------------- | -------- | ----------------------------------------------- |
| **Name**            | Yes      | Your project or tool name                       |
| **Category**        | Yes      | Pick from the list, or propose a new one        |
| **Description**     | Yes      | Short description of what the project does      |
| **URL**             | Yes      | Link to your project's website                  |
| **Emoji**           | No       | An emoji to represent your project              |
| **Image URL**       | No       | Logo or icon URL (alternative to emoji)         |
| **Guide URL**       | No       | Link to docs or a getting-started guide         |
| **Product Images**  | No       | Up to 3 screenshot URLs                         |
| **NS Profile URLs** | No       | Up to 3 NS profile links for the team           |
| **Plans**           | No       | Pricing plans with name, price, and feature list |
