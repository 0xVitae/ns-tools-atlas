# Submitting a Project

Any NS member can submit their project to the Atlas. Submissions are reviewed before going live.

## How to submit

1. Open the Atlas at `/`
2. Click the **+** button
3. Verify your NS profile URL
4. Fill in your project details and submit

:::tip Quick submit
The only required fields are **Name** and **Category** — you can always add more details later.
:::

## Project fields

| Field               | Required | Description                                |
| ------------------- | -------- | ------------------------------------------ |
| **Name**            | Yes      | Your project or tool name                  |
| **Category**        | Yes      | Pick from the list, or propose a new one   |
| **Description**     | No       | Short description of what the project does |
| **URL**             | No       | Link to your project's website             |
| **Guide URL**       | No       | Link to docs or a getting-started guide    |
| **Emoji**           | No       | An emoji to represent your project         |
| **Image URL**       | No       | Logo or icon URL                           |
| **Product Images**  | No       | Up to 3 screenshot URLs                    |
| **NS Profile URLs** | No       | Up to 3 NS profile links for the team      |

## Requesting a tool

Don't have a project but want to see something built? Head to `/requests` to suggest an idea. Other members can upvote requests to signal demand.
