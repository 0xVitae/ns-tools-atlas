# How to Set Up Authentication

To submit a project, you need to verify your identity through your NS profile. This page walks you through getting set up.

## Creating your NS account

If you're not already a Network School member:

1. Go to [ns.com](https://ns.com) and sign up
2. Complete your profile — make sure your profile URL is public

## Verifying your profile

When you click **+** to submit a project on the Atlas, you'll be prompted to enter your NS profile URL. The Atlas validates this URL to confirm you're a real NS member before letting you proceed.

:::info Profile URL format
Your profile URL looks like: `https://ns.com/profile/your-username`
:::

For full platform documentation and account setup, visit the [NS Platform Docs](https://ns.com/platform/docs).
